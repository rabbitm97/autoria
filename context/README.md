## Resumo executivo

**Autoria** é uma plataforma SaaS de autopublicação com IA nativa para o mercado brasileiro.

- **Produto:** Autor entra com manuscrito → sai com livro físico, eBook e audiolivro publicados em 15+ plataformas
- **Diferencial:** Claude como motor editorial (revisão, sinopse, capa IA via DALL-E 3, audiolivro via ElevenLabs) — em português, com preço acessível
- **Stack:** Next.js 14 + Supabase + Claude API + DALL-E 3 + ElevenLabs + Draft2Digital
- **Modelo:** Fundador solo + 9 agentes Claude especializados como equipe
- **Meta Ano 1:** R$165k em receita, 1.000 títulos publicados, fluxo 100% automatizado
- **Janela:** Spines (líder mundial) ainda não opera em português — 12–18 meses de vantagem
